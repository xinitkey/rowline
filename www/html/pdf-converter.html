Description

This tool allows you to convert documents to PDF, split PDF files into multiple parts, or merge multiple PDFs into one. Select the operation above and upload your files.

  • Convert to PDF: Transform various file formats (HTML, XML, XLSX, DOCX, images, text files) into PDF.
  • Split PDF: Extract specific pages or split each page into separate files.
  • Merge PDFs: Combine multiple PDF files into a single document.

Supported input formats for conversion: PDF, HTML, XML, XLSX, XLS, DOCX, JPG, PNG, BMP, TIFF, TXT, PY, LOG, MD
