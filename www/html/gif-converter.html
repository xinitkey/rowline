Description

This tool allows you to convert video files (MP4, WebM, AVI, etc.) to animated GIF format.

  • Start/End Time: Trim your video to create GIFs from specific segments.
  • Frames Per Second: Control animation smoothness and file size (15-30 fps recommended).
  • Output Width: Resize the GIF to reduce file size (optional).

Tips:

  • Shorter videos and lower FPS values produce smaller GIFs.
  • Reducing the output width can significantly decrease file size.
  • Use 10-15 fps for smooth animations with smaller file sizes.
