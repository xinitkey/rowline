Description

This tool extracts tables from PDF files and converts them to Excel (.xlsx) format using the Camelot library. Each detected table is placed in a separate sheet in the output Excel file.

  • Auto-detect: Automatically determines the best extraction method for your PDF
  • Lattice mode: Best for tables with visible grid lines and borders
  • Stream mode: Best for tables without visible borders, relies on whitespace

Tip: Use the "Analyze PDF First" button to preview detected tables before conversion.
